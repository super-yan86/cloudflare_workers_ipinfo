# cloudflare_workers_ipinfo
1、简单版</br>
用Cloudflare_workers搭建一个显示当前网络IP信息的网页，不需要服务器和规则，直接粘贴到worker.js部署即可。</br>
2、复杂版</br>
需要服务器和配置Cloudflare规则，将php文件夹内的文件复制到网站根目录，并配置如图所示Cloudflare规则即可。</br>
